# 职业规划平台 - 技术架构文档

## 项目概述

职业规划平台是一个面向学生的全栈职业发展辅助系统，提供简历管理、职位匹配、AI智能问答、职业评估等功能。系统采用前后端分离架构，包含Java后端服务、Vue3前端应用和Python AI服务三个主要模块。

## 系统架构

系统整体采用微服务架构风格，通过HTTP/REST API进行模块间通信：

```
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   Vue3 Frontend │◄────►│  Spring Backend │◄────►│   FastAPI AI    │
│                 │      │                 │      │                 │
│  - 用户界面     │      │  - 业务逻辑     │      │  - 简历解析     │
│  - 状态管理     │      │  - 数据持久化   │      │  - 职位匹配     │
│  - 路由导航     │      │  - 认证授权     │      │  - AI对话       │
└─────────────────┘      │  - 支付集成     │      │  - 报告生成     │
                         │  - 文件处理     │      │  - 图数据库     │
                         └────────┬────────┘      └─────────────────┘
                                  │
                         ┌────────▼────────┐
                         │   中间件组件    │
                         │                 │
                         │  - MySQL        │
                         │  - Redis        │
                         │  - RabbitMQ     │
                         │  - 阿里云OSS    │
                         └─────────────────┘
```

## 核心模块

### 1. 用户认证模块

- **功能**: 用户注册、登录、密码重置、JWT令牌管理
- **技术实现**:
  - BCrypt密码加密
  - JWT令牌生成与验证
  - Redis会话存储

### 2. 简历管理模块

- **功能**: 简历创建、编辑、导出、模板选择
- **技术实现**:
  - 阿里云OSS文件存储
  - Word/PDF文档生成 (POI-tl, OpenHTMLToPDF)
  - 多格式简历导出支持

### 3. 职位匹配模块

- **功能**: 基于简历的智能职位推荐
- **技术实现**:
  - Python AI服务提供匹配算法
  - Spring Boot与Python服务HTTP通信
  - 异步处理与缓存优化

### 4. AI智能助手

- **功能**: 职业规划咨询、面试问题生成
- **技术实现**:
  - FastAPI构建REST API
  - 图数据库血缘追踪
  - 流式响应支持

### 5. 会员与支付

- **功能**: 会员订阅、积分充值、支付宝支付
- **技术实现**:
  - 支付宝SDK集成
  - 积分系统与交易记录
  - RabbitMQ异步消息处理

### 6. 面试管理

- **功能**: 面试日历、回顾记录
- **技术实现**:
  - 日历可视化 (ECharts)
  - 数据持久化与查询

## 部署配置

### 开发环境

- 后端: `application-dev.yaml`
- 前端: Vite Dev Server (端口5173)
- AI服务: FastAPI (端口8000)

### 生产环境

- 后端: `application-prod.yaml`
- 前端: Vite Build (静态文件)
- Nginx反向代理

## 安全特性

- JWT无状态认证
