# 数据库设计文档

## 概述

本文档描述职业规划平台的数据库表结构设计。数据库采用MySQL，使用MyBatis Plus作为ORM框架。

---

## 一、用户模块

### 1.1 用户表 (user_stu)

存储用户基本信息。

| 字段名 | 类型 | 长度 | 可空 | 默认值 | 说明 |
|--------|------|------|------|--------|------|
| id | BIGINT | - | 否 | 自增 | 用户ID |
| username | VARCHAR | 50 | 是 | - | 用户名 |
| nickname | VARCHAR | 50 | 是 | - | 用户昵称 |
| email | VARCHAR | 100 | 是 | - | 邮箱 |
| password | VARCHAR | 255 | 是 | - | 密码(BCrypt加密) |
| avatar | VARCHAR | 500 | 是 | - | 头像URL |
| phone | VARCHAR | 20 | 是 | - | 手机号 |
| role | INT | - | 是 | 1 | 角色: 1-学生 2-指导老师 3-管理员 |
| status | INT | - | 是 | 1 | 账号状态: 0-禁用 1-启用 |
| info | TEXT | - | 是 | - | 个人简介 |
| is_deleted | INT | - | 是 | -1 | 逻辑删除: -1删除 |
| create_time | DATETIME | - | 是 | - | 注册时间 |
| update_time | DATETIME | - | 是 | - | 更新时间 |

| graduation_year | INT | - | 是 | - | 毕业年份 |
| resume_url | VARCHAR | 500 | 是 | - | 简历文件URL |
| create_time | DATETIME | - | 是 | - | 创建时间 |
| update_time | DATETIME | - | 是 | - | 更新时间 |

## 四、索引设计

### 4.1 常用查询索引

```sql
-- 用户表索引
CREATE INDEX idx_user_email ON user_stu(email);
CREATE INDEX idx_user_username ON user_stu(username);
CREATE INDEX idx_user_status ON user_stu(status);

-- 积分交易索引
CREATE INDEX idx_points_user_id ON points_transaction(user_id);
CREATE INDEX idx_points_create_time ON points_transaction(create_time);

-- 支付订单索引
CREATE INDEX idx_order_user_id ON payment_order(user_id);
CREATE INDEX idx_order_status ON payment_order(status);

-- 面试记录索引
CREATE INDEX idx_interview_user_id ON user_interview(user_id);
CREATE INDEX idx_interview_time ON user_interview(interview_time);
```
