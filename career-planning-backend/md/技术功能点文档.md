## 系统架构

### 技术栈概览
```
前端 ←→ Spring Boot REST API ←→ AI服务(Python)
                ↓
          MySQL数据库
                ↓
        Redis缓存/验证码
                ↓
      阿里云OSS存储/支付宝
```

### 核心技术选择
- **后端框架**: Spring Boot 3.5.9 + Java 21
- **ORM框架**: MyBatis-Plus (自动CRUD)
- **数据库**: MySQL (关系型存储)
- **缓存**: Redis (会话/验证码)
- **消息队列**: RabbitMQ (异步任务)
- **文件存储**: 阿里云OSS
- **支付集成**: 支付宝SDK
- **认证**: JWT Token
- **AI对接**: WebFlux异步HTTP客户端

### 数据流设计
```
Request → Controller → Service → Mapper → Database
   ↓         ↓            ↓
 Route    业务逻辑      ORM操作
                ↓
          ThreadLocal(用户信息)
                ↓
         Response JSON
```

## 安全与性能

### 安全机制
- ✅ **密码加密**: BCrypt安全加密存储
- ✅ **Token认证**: JWT + Refresh Token机制
- ✅ **防枚举攻击**: 登录失败统一错误提示
- ✅ **验证码限流**: 60秒内同一邮箱只能发送一次
- ✅ **支付验证**: 支付宝回调签名验证

### 性能优化
- 🚀 **缓存策略**: Redis缓存热数据
- 🚀 **异步处理**: RabbitMQ处理耗时任务
- 🚀 **流式响应**: SSE流式推送，降低内存占用
- 🚀 **连接池**: MySQL连接池复用
