## API接口体系

### 用户相关
- POST /user/register - 注册
- POST /user/login - 登录
- GET /user/info - 获取信息
- PUT /user/edit - 编辑资料
- POST /user/send-code - 发送验证码
- GET /user/avatar - 上传头像

### 对话相关
- POST /chat/message - 聊天消息
- POST /chat/stream - 流式对话
- POST /chat/upload - 文件上传

### 匹配相关
- POST /matching/job - 职位匹配

### 规划相关
- POST /report/plan - 成长计划
- POST /report/check - 报告检查
- POST /report/polish - 内容润色

### 代码评估
- POST /codeAbility/evaluate - 代码评估

### 知识图谱
- POST /knowledge-graph/analyze/stream - 知识分析
- POST /knowledge-graph/explain/stream - 知识讲解

### 简历导出
- POST /api/resume/export/{format} - 简历导出

### 支付相关
- POST /alipay/pay - 创建订单
- GET /alipay/return - 支付回调

### 积分相关
- GET/POST /points/* - 积分管理
- GET/POST /transaction/* - 交易记录
